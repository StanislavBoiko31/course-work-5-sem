# Інформація про зберігання файлів

## Де зберігаються файли результатів роботи?

Файли, які завантажує майстер (фото та відео), зберігаються в наступній структурі:

```
backend/
  media/
    booking_results/
      {booking_id}/
        photos/
          photo1.jpg
          photo2.jpg
          ...
        videos/
          video1.mp4
          video2.mp4
          ...
```

### Приклад:
Якщо майстер завантажує результати для замовлення з ID=5, файли будуть зберігатися в:
- `backend/media/booking_results/5/photos/` - для фото
- `backend/media/booking_results/5/videos/` - для відео

### Налаштування Django:

- **MEDIA_ROOT**: `backend/media/` (визначено в `studio/settings.py`)
- **MEDIA_URL**: `/media/` (визначено в `studio/settings.py`)

### Доступ до файлів:

Файли доступні через URL: `http://127.0.0.1:8000/media/booking_results/{booking_id}/photos/{filename}`

### Важливо:

1. Папка `media/` автоматично створюється при першому завантаженні файлу
2. Файли зберігаються на сервері і доступні для завантаження клієнтами
3. При видаленні замовлення файли НЕ видаляються автоматично (потрібно додати логіку очищення)
